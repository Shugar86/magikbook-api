# Scripts — утилиты управления данными

Запускать из корня проекта:

```bash
# Засеять базу тестовыми данными
python -m scripts.seed_db

# Засеять текстовые промпты
python -m scripts.seed_text_prompts
```

## Требования

- Python 3.11+
- Установленные зависимости проекта (`pip install -e .`)
- Настроенная база данных (через `.env`)

## Примечания

- Скрипты используют абсолютные импорты (`from src.models...`)
- Для корректной работы PYTHONPATH должен включать корень проекта
- При запуске через `python -m` из корня проекта импорты работают автоматически
