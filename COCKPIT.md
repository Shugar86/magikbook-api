# COCKPIT — magikbook-api

## Who this project is (personality, 1-2 sentences)

MagikBook API — **магический движок за кулисами**: серьёзный инженерный бэкенд с игровой душой (мана, гримуар, битва промптов, «заклинание дня»). Не холодный CRUD, а инфраструктура сообщества создателей AI-контента.

## How it feels (tone, vibe — for each layer/persona)

| Слой | Vibe |
|------|------|
| **API для фронта** | Надёжный, предсказуемый — CORS, cookie-сессии, чёткие 404/409; «можно строить UI и не гадать». |
| **Модерация / админ** | Контроль и ответственность — очередь, approve/reject, fallback превью, партнёрские ссылки. |
| **Битва / ELO** | Соревновательный азарт — честные голоса, анти-накрутка, рейтинг как в играх. |
| **Генерация (Gemini)** | Потоковая магия — SSE-стрим, лимиты маны, fallback моделей. |
| **VK / Telegram** | Прагматичная дистрибуция — постинг «куда живёт аудитория», с боями против лимитов платформ. |
| **Dev / CI** | Профессиональная дисциплина — ruff, pytest, GitHub Actions, чеклисты prod; без сюрпризов с prod DB в git. |

## For whom (3 audiences)

1. **Авторы промптов** — загрузка, публикация, портфолио, гримуар, битва за место в ленте.
2. **Модераторы / админы** — очередь контента, статистика кабинета, ручной постинг при сбое автопоста.
3. **Команда MagikBook (full-stack + ops)** — единый бэкенд для Next.js-фронта, Docker Compose, Redis, миграции Alembic.

## Emotions it evokes

- **Для автора:** гордость за работу в ленте, азарт битвы, удовлетворение от «ожившего» промпта в VK/TG.
- **Для модератора:** спокойствие от контроля очереди; раздражение при сбоях VK — mitigated документацией и fallback.
- **Для разработчика:** уверенность от async-стека и тестов; бдительность из-за внешних API и ops-инцидентов (SQLite, токены).

## What makes it special (3-5 unique traits)

1. **Битва промптов с real ELO** — не декоративные лайки, а pairwise voting + пересчёт рейтинга.
2. **Полный цикл контента** — upload → moderation → feed → social publish (VK + Telegram) в одном API.
3. **Мульти-OAuth + email OTP + Telegram Login** — вход «как удобно пользователю».
4. **SSE-генерация промптов** — живой стрим через Gemini, не блокирующий JSON.
5. **Игровая экономика «маны»** — daily bonus, tokens, paywall/Telegram Stars — продуктовый слой поверх CRUD.

## Current focus (one paragraph, link to STATE.md for blockers)

Сейчас проект **дошлифовывает дистрибуцию и идентичность автора**: VK-постинг стабилизирован (form body, токены, cookie auth), параллельно в рабочей копии собирается **Artist Glory** — публичное портфолио (`/api/users/{username}/portfolio`) и связанные поля в ленте/загрузках. Фоновая генерация «заклинания дня» временно выключена. Детали статуса, незакоммиченных изменений и блокеров — в [STATE.md](STATE.md).
